[YouTrack](http://jetbrains.com/youtrack) Version Manager inspired by [nvm](https://github.com/creationix/nvm)


## Installation

Before install *yvm* you should have installed *node* and *npm*.

```sh
npm install -g yvm
```


# Usage


```sh
yvm help
```
